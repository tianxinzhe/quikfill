🚀 快速开始 / Getting Started

1

右键网页选中文本 | Select Text on Webpage

在网页上选中任意文字，然后 右键 点击，选择 "📥 收藏到 QuickFill"
Select any text on a webpage, then right-click and choose "📥 Save to QuickFill"

2

选择分类并保存 | Choose Category & Save

在弹出的对话框中选择分类（可多选），点击确认保存
Select categories (multi-select allowed) in the popup dialog and click confirm to save

3

点击文本快速填充 | Click to Fill

打开侧边栏，点击任意文本即可自动填充到网页输入框中
Open the sidebar, click any text to auto-fill into webpage input fields

📁 分类管理 / Category Management

如何添加分类？ / How to Add Category?

  1. 点击分类栏右侧的 + 按钮 | Click the + button next to categories
  2. 输入分类名称（最多5个字符）| Enter category name (max 5 characters)
  3. 点击确定完成创建 | Click OK to create

如何删除分类？ / How to Delete Category?

  1. 点击分类栏右侧的 ⚙️ 按钮进入管理模式 | Click ⚙️ to enter manage mode
  2. 点击要删除分类旁边的 × | Click × next to the category
  3. 在确认对话框中点击确定 | Confirm in the dialog

💡 删除分类后，该分类下的文本会变为「未分类」状态，不会丢失
💡 Texts under deleted category become "uncategorized", not deleted

🔧 变量说明 / Variables

在文本中使用变量，填充时会自动替换为实际值。变量格式：{{变量名}}
Use variables in texts, they will be replaced with actual values when filling. Format: {{variable}}

📋 批量操作 / Batch Operations

1

点击「编辑」按钮 | Click "Edit"

进入编辑模式，显示复选框和批量操作按钮
Enter edit mode, checkboxes and batch buttons appear

2

选择要删除的文本 | Select Texts

点击文本左侧的复选框，或使用「全选」按钮
Click checkboxes or use "Select All"

3

点击「删除」按钮 | Click "Delete"

确认删除后，选中的文本将被永久移除
Confirm to permanently delete selected texts

📝 使用方法 / How to Use

  1. 点击下载按钮保存JSON文件到本地 | Click download to save JSON file
  2. 打开QuickFill侧边栏 | Open QuickFill sidebar
  3. 点击「导入」按钮 | Click "Import" button
  4. 选择下载的JSON文件 | Select the downloaded JSON file
  5. 选择「覆盖」或「合并」导入方式 | Choose "Overwrite" or "Merge"
